# PromptCraft
    
    ## Purpose
    The app was built to help a friend easily generate highly specific, high-fidelity prompts for AI photo restoration and upscaling. It solves the common issue of AI altering or hallucinating details in photos by ensuring the prompt provides sufficient specificity regarding constraints and output sizes.
    
    ## Instructions
    1. Ensure Python is installed.
    2. Install Streamlit (`pip install streamlit`).
    3. Run the app (`streamlit run PromptCraft.py`).
